<Aside type="tip">
[fx.go](https://github.com/denful/fx.go) is an algebraic effects system for Golang, an ability and effect-handler system that brings structured effects to Go.
</Aside>

## What it does

fx.go provides an ability system and algebraic effect handlers for Go: declare effects, interpret them with handlers, and compose effectful code with explicit, typed capabilities rather than ad-hoc side effects.

## Role in the ecosystem

fx.go is the Go member of a cross-language algebraic-effects family alongside [nfx](/ecosystem/nfx/) (Nix) and [fx-rs](/ecosystem/fx-rs/) (Rust). fx-rs is directly modeled after it.
